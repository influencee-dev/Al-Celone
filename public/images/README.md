# Directory per le immagini dell'Agriturismo Al Celone
Questa cartella è destinata a contenere tutte le foto del sito (es. logo, sfondi, foto delle camere e dei piatti).
I file all'interno di questa cartella verranno pubblicati direttamente nella root del sito sotto il percorso `/images/`.

Ad esempio, se carichi un'immagine chiamata `home-hero.jpg` in questa posizione, nel codice sarà accessibile tramite `/images/home-hero.jpg`.
